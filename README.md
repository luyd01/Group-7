🧱 Folder Structure

components/ → Reusable components or modules

docs/ → Documentation files

🏗️ Architecture Approach



We used a simple layered structure:



Separation of core logic (src)

Modular components (components)

Supporting documentation (docs)



This approach improves:



Maintainability

Scalability

Readability of the project
